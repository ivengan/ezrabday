# Ezra's Birthday (ezrabday)

A premium, interactive web application to celebrate Ezra's special day.

## Features
- **Modern Design**: A stunning, vibrant, and responsive user interface.
- **Dynamic Interactions**: Smooth animations and transitions.
- **Micro-Animations**: Enhanced user engagement through subtle visual cues.

## Getting Started
To view the project locally, simply open `index.html` in your browser or run a local development server:
```bash
npm run dev
```

## Built With
- HTML5
- Vanilla CSS
- JavaScript
